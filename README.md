# SpringBoot-Demo2.0x  
SpringBoot Demo Project, joins other functions need in real big company  
  
集成功能包括： 
1.Maven管理工具  
2.Mysql数据库（druid数据源）  
3.Mybatis映射  
4.Redis集群  
5.RabbitMQ集群（单点、订阅）  
6.基于RabbitMQ的延时队列  
7.定时任务  
8.多环境配置切换与部署  
9.Logback日志输出  
10.Thymeleaf与jsp模板 
11.自定义异常和全局异常捕捉  
12.自定义过滤器、拦截器（api入参出参自动记录、耗时统计、日志追踪等）  
13.文件上传与下载  
14.swagger-ui插件
